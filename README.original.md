# AVTOPLAKAT HANDOFF

Этот каталог предназначен для открытия как локального проекта в Antigravity.

Содержимое:
- `AUDIT.md` — первый технический аудит.
- `data/content.json` — опубликованный контент WordPress с метаданными и привязками изображений.
- `data/terms.json` — категории и таксономии.
- `theme/` — старая кастомная тема для анализа логики и полей; файлы шрифтов исключены.
- `uploads/` — изображения и медиа.

Не восстанавливай WordPress и не переписывай исходные данные. Сначала изучи структуру и подготовь план нового Astro-сайта для украинского рынка.
