# Поточний статус проєкту AVTOPLAKAT (CURRENT_STATE)

Цей файл містить опис архітектури, поточного стану реалізації та наявного коду станом на липень 2026 року.

---

## 1. Стек технологій
* **Фронтенд:** Astro (статична генерація SSG).
* **Стилі:** Tailwind CSS v4 (із інтеграцією через `@tailwindcss/vite` плагін у Vite/Astro).
* **Логіка клієнта:** Vanilla JS / TypeScript (без jQuery, React чи інших важких JS-фреймворків).
* **Бекенд:** Cloudflare Pages Functions (Serverless API).
* **Валідація:** Zod (сувора перевірка на фронтенді та бекенді).
* **Хостинг:** GitHub + Cloudflare Pages.

---

## 2. Структура проєкту та модулі
Проєкт розроблено за модульною архітектурою (KISS):

* `/src/content/products/` — База даних товарів (112 файлів JSON).
* `/src/types/schemas.ts` — Zod-схеми для типізації.
* `/src/config/defaults.ts` — Глобальна конфігурація цін та характеристик плакатів.
* `/functions/services/` — Незалежні модулі інтеграцій (Telegram, Monobank, Nova Poshta).
* `/functions/api/order.ts` — Серверна точка обробки замовлення.
* `/src/styles/global.css` — Конфігурація теми Tailwind v4 (`@theme`) та базові утиліти.

---

## 3. Результати розробки та перевірок
* **Стилізація:** Всі компоненти та сторінки переведені на чистий Tailwind CSS v4. Vanilla CSS стилі видалено.
* **UX Спрощення (Design Philosophy v1.0):**
  * Додано великий клікабельний телефон у шапці сайту (`Header.astro`).
  * Email зроблено необов'язковим для фізичних осіб на сторінці оформлення замовлення.
  * Індивідуальні вимоги у кошику об'єднано у велике глобальне поле коментаря під списком товарів замість перемикачів біля кожного товару.
  * Видалено дублюючі блоки характеристик на сторінці товару.
  * Видалено зайві блоки та дрібні іконки хрестиків видалення товарів (замінено текстовою кнопкою "Видалити").
* **Збірка:** Проєкт успішно будується (`npm run build`) за 1.3 секунди, генеруючи 123 чисті статичні сторінки.
